## 🚀 Current project

I am currently working on my Bachelor's thesis:

**Quantitative Modeling of Energy Consumption and Carbon Impact of AI Inference Workloads**

The project models GPU-cluster energy consumption and CO₂ emissions for AI inference workloads using Python-based simulation. It combines workload modeling, queueing theory, GPU power states, datacenter PUE, and electricity carbon intensity.

Main topics:
- AI inference workload modeling
- GPU cluster utilization
- Energy and carbon footprint estimation
- Python simulation and data analysis
- Queueing models such as M/M/c and G/M/c
